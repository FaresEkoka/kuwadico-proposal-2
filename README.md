# Proposal Microsite

A premium, cinematic single-page proposal website built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Full-screen frames**: Each section is 100vh with scroll-snap
- **Smooth animations**: Framer Motion animations on scroll
- **Dark, minimal design**: Charcoal/near-black base with muted neon-green accent
- **Progress indicator**: Top-right frame counter (01 / 08)
- **Navigation dots**: Left-side nav for quick frame jumping
- **Responsive**: Works beautifully on desktop and mobile
- **Content-driven**: All copy in `content.ts` for easy editing

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

Edit `content.ts` to update all copy, client information, and pricing. The site will automatically reflect your changes.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
